Welcome to Miru!

Miru (見る) - to look, to see, to watch.

Miru is a website monitoring tool for the Government Data Archiving project. Miru works with a simple three-stage process.

  • You make a request to have a site monitored.
  • An archiver reviews the request, telling the tool to start monitoring the site.
  • Miru will then start to monitor the site for changes and inform archivers of updates.
